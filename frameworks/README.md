# Frameworks

Hands-on learning and experimentation with popular frameworks across different domains.

## Categories

### Web Frameworks

**Frontend**
- React - Component-based UI library
- Vue.js - Progressive JavaScript framework
- Angular - Full-featured TypeScript framework
- Svelte - Compile-time framework
- Next.js - React framework for production

**Backend**
- Express.js - Minimalist Node.js framework
- Django - Python web framework
- Flask - Lightweight Python framework
- FastAPI - Modern Python API framework
- Spring Boot - Java enterprise framework
- ASP.NET Core - C# web framework

### Mobile Frameworks
- React Native - JavaScript for iOS/Android
- Flutter - Dart framework for cross-platform
- Swift/SwiftUI - iOS native development
- Kotlin/Jetpack Compose - Android native

### Backend & API
- GraphQL - Query language for APIs
- gRPC - High-performance RPC framework
- REST APIs - RESTful design patterns
- WebSockets - Real-time communication

### Testing Frameworks
- Jest - JavaScript testing
- Pytest - Python testing
- JUnit - Java testing
- Selenium - Browser automation
- Cypress - E2E testing
- Mocha/Chai - JavaScript test frameworks

### ML/AI Frameworks
- TensorFlow - End-to-end ML platform
- PyTorch - Deep learning framework
- scikit-learn - Classical ML algorithms
- Keras - High-level neural networks API
- Hugging Face - NLP transformers

## Learning Strategy

1. **Understand fundamentals** - Learn the language first
2. **Follow tutorials** - Official documentation is best
3. **Build small projects** - Todo apps, API servers, etc.
4. **Study best practices** - Folder structure, patterns
5. **Read real code** - Explore open source projects
6. **Deploy something** - Get it running in production
