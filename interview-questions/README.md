# Interview Questions

A collection of technical interview questions and solutions organized by category.

## Categories

### Algorithms
Common algorithmic problems including:
- Sorting algorithms (QuickSort, MergeSort, HeapSort)
- Searching algorithms (Binary Search, DFS, BFS)
- Dynamic Programming
- Greedy Algorithms
- Backtracking
- Divide and Conquer

### Data Structures
Implementation and problems involving:
- Arrays and Strings
- Linked Lists
- Stacks and Queues
- Trees (Binary Trees, BST, AVL, Red-Black)
- Graphs
- Hash Tables
- Heaps
- Tries

### System Design
Design scalable systems:
- URL Shortener
- Social Media Feed
- Messaging System
- Rate Limiter
- Cache System
- Load Balancer
- Database Sharding
- Microservices Architecture

### Behavioral
Practice common behavioral questions:
- Leadership and influence
- Problem-solving approach
- Conflict resolution
- Project experiences
- Failure and learning
- Team collaboration

### Coding Challenges
Platform-specific challenges from:
- LeetCode (Easy, Medium, Hard)
- HackerRank
- CodeWars
- Project Euler
- Advent of Code

## Study Plan

1. **Week 1-2**: Arrays and Strings (Easy)

3. **Week 5-6**: Trees and Recursion (Medium)
4. **Week 7-8**: Graphs and Dynamic Programming (Medium-Hard)
5. **Week 9-10**: System Design Fundamentals
6. **Week 11-12**: Mock Interviews and Review
